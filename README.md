# employee-postgrelsql
 📊 Employee Data Analysis using PostgreSQL

📌 Project Overview
This project focuses on analyzing employee data using PostgreSQL to uncover insights related to employee demographics, experience, salary tiers, and attrition patterns.



 📂 Dataset Description
The dataset contains the following columns:
- Education  
- JoiningYear  
- City  
- PaymentTier  
- Age  
- Gender  
- EverBenched  
- ExperienceInCurrentDomain  
- LeaveOrNot  



 Objectives
- Analyze employee distribution across cities and education levels  
- Understand the relationship between experience and salary (PaymentTier)  
- Identify factors affecting employee attrition  
- Explore patterns in benched employees  



 🛠 Tools & Technologies
- PostgreSQL  
- SQL  
- Data Analysis  



🧠 Key SQL Concepts Used
- SELECT, WHERE, ORDER BY  
- GROUP BY, HAVING  
- Aggregate Functions (AVG, COUNT, MAX, MIN)  
- Subqueries  
- Window Functions (RANK)  


📊 Key Insights
- Employees with higher experience tend to have better payment tiers  
- Certain cities show higher retention rates  
- Benched employees show different leaving patterns  
- Age and experience influence attrition
